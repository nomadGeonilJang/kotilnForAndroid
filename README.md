### 001 변수와 자료형
1. var 변경이 가능
2. val 변경이 불가능
3. class 매부 프로퍼티 : 속성
4. 그외 로컬 변수라고 한다.
5. 자료형? 를 사용할경우 null을 혀용하겠다는 의미로 사용된다.

```code

package Kotlin2020

fun main(args : Array<String>){

    var a: Int = 10 // variable : 변할 수 있는 변수 방법 -> let
    val b: Int = 20 // value : 변할 수 없는 변수 선언 방법 -> const

}

```
