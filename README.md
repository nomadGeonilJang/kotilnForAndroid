# Kotlin

### 001 변수와 자료형
```code
package Kotlin2020

fun main(args : Array<String>){

    var a: Int = 10 // variable : 변할 수 있는 변수 방법 -> let
    val b: Int = 20 // value : 변할 수 없는 변수 선언 방법 -> const

}
```
